# Ender 3 Pro LCD Display Case

I own a 3D printer and a long while ago I printed a custom case for the LCD display that controls the printer. Since then, I wanted to make my own and well this is my attempt at it! There are two versions, one without drawers, another with. I actually learned quite a bit this time round since making a robust model is quite difficult, but it is possible. Note that the measurements are approximate and I did not measure them to fit the display yet. 

Time Taken (Total): 5 Hours

## Skills Learned

- How to split models using split in Part Workbench
- Using Part Workbench and Part Design in conjunction
- Using Subspace binders to reference geometry between parts
- Using keybinds to make things faster
- Making screw holes consistent between two different bodies by using a common sketch that is independent of any body.
