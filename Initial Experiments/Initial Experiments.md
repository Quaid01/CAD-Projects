# Initial Experiments

These 3 files basically were my reintroduction into FreeCAD after the 1.1 release. It was a few years since I touched the software (it was prior to the 1.0 version and I couldn't understand it that well), so as a warm up and to refamiliarize myself I made these files.

They are mainly incomplete with no real purpose except practice. The double pendulum was a practice with assembly and sketching, which then failed after I realized that FreeCAD (at least, not trivially) couldn't do inverse kinematics the way I was hoping. However, it did provide a good ground to work on making an assembly which was a success.



## Skills Learned

* Basic Assembly
* Basic Sketching Principles
