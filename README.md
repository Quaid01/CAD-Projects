# CAD Projects

This is a repository containing all of my 3D models that I have created over the years. This is meant to act as a portfolio of how I have improved and as an easy way to share the portfolio if need be. It also documents history changes which is similar to Onshape's history system, which FreeCAD lacks.

The following softwares were used to make the models:

- FreeCAD
- Blender

I wanted to own my models which is why I opted to use open-source software instead of paid software with licenses and restrictions. Not only that, I don't have to pay to make things, which in my opinion is a very nice thing for a broke student.
